# 每日稽核 Daily Auditor

以學習、分享與未來創業為方向的個人知識網站。

## 本機預覽

直接開啟 `index.html`，或使用 VS Code 的 Live Server。

## 發布

網站為純靜態 HTML、CSS 與 JavaScript，可直接部署至 GitHub Pages。

1. 將程式碼推送至 GitHub 儲存庫。
2. 開啟 `Settings > Pages`。
3. 在 `Build and deployment` 選擇 `Deploy from a branch`。
4. 選擇 `main` 與 `/ (root)` 後儲存。
