AI × AUDIT · DAY 07 · 實用工具系列

第一關：召喚你的第一個 AI 稽核助手

從「破案推理」到「動手實作」，這次換你自己下場。五關破完，你會在自己的電腦上跑出第一支採購異常分析程式。

Arthur Agent 編輯室2026 年 7 月 2 日約 12 分鐘閱讀
實際案例八筆虛構採購資料裡藏著兩筆高額交易。你不用先成為工程師，只要把查核需求說清楚，讓 AI 協助寫 Python，再在本機找出 980,000 元與 1,500,000 元的異常。

最後仍由你判斷：這是舞弊線索、控制缺口，還是合理例外。

前六天我們在看推理；今天開始動手。你是主角，AI 是助手，證據才是裁判。

第一關：召喚基礎裝備——安裝 Python

前往 Python 官方下載頁下載 Windows 安裝程式。執行時勾選 Add python.exe to PATH，再完成安裝。打開 PowerShell，輸入：

python --version

看到 Python 與版本號，第一關通過。如果 Windows 找不到指令，也可試試 py --version。版本不是競賽；以官方仍支援、公司允許的版本為準。

第二關：建造基地——安裝 VS Code

  1. 從 VS Code 官方網站下載並安裝。
  2. 在「擴充功能」搜尋 Python，安裝 Microsoft 發布的官方擴充套件。
  3. 建立並開啟 ai-audit-lab 資料夾，再新增 test.py。
print("我的稽核助手基地啟動成功")

按右上角執行鍵，終端機出現這句話，基地正式落成。

第三關：讓 AI 寫出「採購異常偵探」

在資料夾新增 purchase_data.csv，貼入以下虛構資料：

供應商,採購金額,採購日期,經辦人
A公司,150000,2026-01-05,莫里亞蒂
B公司,220000,2026-01-08,雷斯垂德
A公司,980000,2026-01-10,莫里亞蒂
C公司,75000,2026-01-12,華生
A公司,145000,2026-01-15,莫里亞蒂
B公司,210000,2026-01-18,雷斯垂德
D公司,1500000,2026-01-20,莫里亞蒂
A公司,152000,2026-01-22,莫里亞蒂

把這段需求交給核准的 AI

「我是一名稽核人員。請寫一支不連網的 Python 程式：讀取本機 purchase_data.csv；標記金額超過全體平均兩倍的交易；統計每位經辦人的總金額與占比；把結果印出並存成 result.csv。請用繁體中文註解，檢查必要欄位與數字格式，錯誤時顯示可理解的訊息。不得把資料傳送到外部。」

AI 多半會用 pandas 讀取表格、計算平均值、用 groupby 加總，再輸出 CSV。你不必立刻看懂每一行，但一定要確認：規則是不是你要的、檔案讀寫位置是否正確、程式有沒有不必要的網路功能。

第四關：在本機執行，見證結果

把程式存成 audit_analysis.py，在 VS Code 終端機安裝套件並執行：

py -m pip install pandas
py audit_analysis.py

這組資料總額是 3,432,000 元，平均每筆 429,000 元；兩倍門檻為 858,000 元，因此 980,000 元與 1,500,000 元會被標記。莫里亞蒂經手 2,927,000 元，占總額約 85.3%。這是集中風險警示，不是舞弊定論。

異常只是值得追查的線索；合約、驗收、核准與訪談，才可能把線索變成稽核證據。

第五關：紅色警報別慌——請 AI 協助除錯

找不到 pandas

ModuleNotFoundError: No module named 'pandas'

在目前選用的 Python 環境執行 py -m pip install pandas，再重跑程式。

找不到 CSV

FileNotFoundError: purchase_data.csv

確認 CSV 與 Python 檔案在同一資料夾，也確認 VS Code 終端機目前位於 ai-audit-lab。

其他錯誤

複製從 Traceback 到最後一行的錯誤訊息，說明你執行的指令、預期結果與實際結果，再請 AI 解釋原因並提出最小修改。貼出前先刪除真實路徑、帳號、主機名稱、檔名、資料內容、金鑰與其他公司資訊。錯誤訊息也可能洩密；不要把「完整複製」誤會成「未檢查就上傳」。

你的通關檢查表

  • Python 與 VS Code 都能正常執行。
  • 測試資料完全虛構，不包含公司或個人資料。
  • AI 產生的程式只讀指定輸入、只寫指定輸出，而且不連網。
  • 你能說明異常規則、門檻與經辦人集中占比。
  • 你會先去識別化錯誤訊息，再請 AI 協助除錯。
  • 你知道異常標記需要人工驗證，不能直接當作稽核結論。
解決案例

莫里亞蒂被抓到了嗎？還沒有——但你已經找到該查的地方

程式從八筆交易中找出兩筆超過平均兩倍的高額交易，也指出同一經辦人占總採購額 85.3%。下一步是抽出請購、比價、核准、驗收與付款證據，確認是否有拆單、權限繞過、關係人或合理的大額專案。程式縮小範圍；稽核人員完成判斷。
